## What is this repository?
This is a Discord selfbot made in Python specifically for server nuking.

## What is a Discord selfbot?
A selfbot is a bot that runs on a user token rather than a bot token. Only people on the account running it can use the commands.

## Heads up
Selfbots were removed in discord.py after 1.7.3, so this uses discord.py version 1.7.3 to run. Do not use any later versions.
I also strictly advise that you do not use this selfbot, as this was made for educational purposes only. Selfbots violate Discord's Terms of Service.
